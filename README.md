# question--answer-picture-system


# 智能相册管理与问答系统

一个基于AI的照片管理工具，支持批量处理照片生成描述和标签，并通过自然语言搜索照片，帮助你快速找到想要的图片。


## 功能特点
- **批量处理**：自动分析文件夹中的所有照片，生成结构化描述和多维度标签
- **智能搜索**：通过自然语言查询（如"2024年和朋友在海边的照片"）快速找到匹配图片
- **单张测试**：支持上传单张照片进行分析，查看生成的描述和标签


## 环境准备

### 前置条件
- Python 3.8及以上版本
- 有效的通义千问API密钥（需自行申请，用于照片内容分析）
